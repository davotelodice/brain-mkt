# Fundamentos de SEO

> Principios para visibilidad en motores de búsqueda.

---

## 1. Marco E-E-A-T

| Principio | Señales |
|-----------|---------|
| **Experience** | Conocimiento de primera mano, ejemplos reales |
| **Expertise** | Credenciales, profundidad de conocimiento |
| **Authoritativeness** | Backlinks, menciones, reconocimiento de la industria |
| **Trustworthiness** | HTTPS, transparencia, información precisa |

---

## 2. Core Web Vitals

| Métrica | Objetivo | Mide |
|---------|----------|------|
| **LCP** | < 2.5s | Rendimiento de carga |
| **INP** | < 200ms | Interactividad |
| **CLS** | < 0.1 | Estabilidad visual |

---

## 3. Principios de SEO Técnico

### Estructura del Sitio

| Elemento | Propósito |
|----------|-----------|
| Sitemap XML | Ayudar al rastreo |
| robots.txt | Controlar acceso |
| Etiquetas canónicas | Prevenir duplicados |
| HTTPS | Señal de seguridad |

### Rendimiento

| Factor | Impacto |
|--------|---------|
| Velocidad de página | Core Web Vital |
| Mobile-friendly | Factor de ranking |
| URLs limpias | Rastreabilidad |

---

## 4. Principios de SEO de Contenido

### Elementos de Página

| Elemento | Mejor Práctica |
|----------|---------------|
| Title tag | 50-60 caracteres, keyword al frente |
| Meta description | 150-160 caracteres, convincente |
| H1 | Uno por página, keyword principal |
| H2-H6 | Jerarquía lógica |
| Alt text | Descriptivo, no relleno |

### Calidad de Contenido

| Factor | Importancia |
|--------|-------------|
| Profundidad | Cobertura comprehensiva |
| Frescura | Actualizaciones regulares |
| Unicidad | Valor original |
| Legibilidad | Escritura clara |

## 6. Guías de Contenido con IA

### Qué Busca Google

| ✅ Hacer | ❌ No Hacer |
|----------|-------------|
| Borrador IA + edición humana | Publicar contenido IA crudo |
| Agregar insights originales | Copiar sin valor |
| Revisión experta | Omitir verificación de hechos |
| Seguir E-E-A-T | Keyword stuffing |

---

## 7. Factores de Ranking (Priorizados)

| Prioridad | Factor |
|-----------|--------|
| 1 | Contenido de calidad, relevante |
| 2 | Backlinks de sitios de autoridad |
| 3 | Experiencia de página (Core Web Vitals) |
| 4 | Optimización móvil |
| 5 | Fundamentos de SEO técnico |

> **Recuerda:** SEO es un juego a largo plazo. Contenido de calidad + excelencia técnica + paciencia = resultados.

## Ejemplos de Uso

### Ejemplo 1: Mejorar el SEO de tu Página Principal
**Situación**: Quieres que tu página principal aparezca mejor en Google pero no sabes por dónde empezar.

**Cómo usar esta skill**:
1. Di a Cursor: "Quiero mejorar el SEO de mi página principal"
2. Cursor usará esta skill para:
   - Verificar que tienes title tag y meta description optimizados
   - Asegurar que tu H1 contiene tu keyword principal
   - Verificar que las imágenes tienen alt text descriptivo
   - Revisar que el contenido es de calidad y relevante
   - Sugerir mejoras basadas en principios E-E-A-T

**Resultado**: Una página optimizada para SEO que tiene mejor oportunidad de rankear en Google.

### Ejemplo 2: Entender por qué tu Contenido No Rankea
**Situación**: Escribes contenido pero no aparece en los resultados de búsqueda.

**Cómo usar esta skill**:
1. Di a Cursor: "Escribo contenido pero no rankea, ¿qué estoy haciendo mal?"
2. Cursor evaluará:
   - Si tu contenido sigue principios E-E-A-T (experiencia, expertise, autoridad, confianza)
   - Si tu sitio es técnicamente sólido (velocidad, mobile-friendly)
   - Si el contenido es mejor que lo que ya rankea
   - Qué factores de ranking necesitas mejorar

**Resultado**: Entendimiento claro de qué mejorar para que tu contenido tenga mejor oportunidad de rankear.

### Ejemplo 3: Optimizar para Core Web Vitals
**Situación**: Google te dice que tu sitio es lento y quieres mejorarlo.

**Cómo usar esta skill**:
1. Di a Cursor: "Quiero mejorar los Core Web Vitals de mi sitio"
2. Cursor te guiará en:
   - LCP: Optimizar la carga de imágenes y recursos principales
   - INP: Mejorar la interactividad de botones y formularios
   - CLS: Prevenir cambios de layout durante la carga
   - Herramientas para medir y monitorear

**Resultado**: Un sitio más rápido que cumple con los estándares de Google y mejora la experiencia del usuario.
